UD316

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UD316 - Search the equatorial point with the lowest magnetic field intensity

Synopsis 

      TYPE(zpnt) mequat
      integer*4 ifail

      CALL UD316(mequat, ifail)

Arguments

    mequat
    equatorial point with the lowest B [out]
    ifail
    error flag (see diagnostics) [out]

Description

    The subroutine UD316 scans the field line segments of a drift shell and determines the point with the lowest magnetic field intensity. The drift shell has to be defined as a set of field line segments stored in the common blocks UC110, UC120, and UC130.

Diagnostics

    -31601, problem with the variable k1st of the common block UC110
    -31602, problem with the variable nbrfl of the common block UC120

Common Blocks

    UC110, magnetic drift shell description
    UC120, magnetic field line description

Dependencies

Called by
