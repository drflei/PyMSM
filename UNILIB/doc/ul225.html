UL225

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UL225 - evaluate invariant radius and latitude

Synopsis 

      INTEGER*4     ifail, nfbm
      REAL*8        fbm(nfbm), flm(nfbm)
      REAL*8        frd(nfbm), fla(nfbm)
      
      CALL UL225 (fbm, flm, nfbm, frd, fla, ifail)

Arguments

    fbm
    Magnetic field intensity at the mirror point [in]
    flm
    McIlwain's shell parameter [in]
    nfbm
    Number of values [in]
    frd
    Invariant radius [out]
    fla
    Invariant latitude (lambda) [out]
    ifail
    Error flag (see diagnostics) [out]

Description

    The subroutine UL225 computes magnetic invariant radius and latitude from the magnetic coordinates Bm and L.

History

    The subroutine is not included in version 2.02 and earlier.

Diagnostics

    -22501, Unable to evaluate, at least, one value

Common Blocks

    UC140, magnetic field description
    UC160, general constants
    UC190, control parameters, set 1

Dependencies

Called by

    None

Calls

    None

See also

    UL220, get information on a magnetic field line segment

Reported Bugs

    None

Examples

    None
