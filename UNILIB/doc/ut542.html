UT542

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UT542 - convert spherical vector components to cartesian components

Synopsis 

      TYPE(zgeo) mpos
      TYPE(zvec) mvec
      TYPE(zxyz) mxyz

      CALL UT542 (mpos,mvec,mxyz)

Arguments

    mpos
    Geographic position [in]
    mvec
    spherical components [in]
    mxyz
    cartesian components [out]

Description

    The subroutine UT542 converts the spherical components of a vector into cartesian components. The transformation is function of the geographic position where it is evaluated.

History

    The subroutine UT542 does not exist in version 1.05 and earlier.

Diagnostics

    None

Common Blocks

    UC160, general constants

Dependencies

Called by

    None

Calls

    None

See also

    UT547, convert cartesian vector components to spherical components
