Question

    G.02 - Where to find the UNILIB library and documentation ?

Answer

    The UNILIB libray is freely accessible in the form of Fortran sources.

    The documentation of the library is provided in the form of HTML pages. It contents:

      FAQ, frequently asked questions
        General questions
        Technical questions
      TOC, list of the components
        Structures
        Common blocks
        Subroutines
        Examples
      Index, search index
      History of the library
