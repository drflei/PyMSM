[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UA635 - evaluate the Debye length

Synopsis 

      REAL*8    dens(30), debye, temp(3)
      INTEGER*4 nsp, ksp(30), ifail
      
      CALL UA635 (dens, nsp, ksp, temp, debye, ifail)

Arguments

    dens
    Number density (see note of subroutine UA630) [in]
    nsp
    Number of species [in]
    ksp
    List of species code [in]
    temp
    Temperature [in]
    debye
    Debye length [out]
    ifail
    Error flag (passed by dependents) [out]

Description

    The subroutine UA635 evaluates the Debye length from the number density and temperature of the ions (dens(20...30) and temp(3), resp.). Note that only the species listed in argument ksp are taken into account.

Diagnostics

    None
