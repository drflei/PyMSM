UM532

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UM532 - external magnetic field

Synopsis 

       REAL*8 r, sla, cla, slo, clo, dbr, dbt, dbp
       
       UM532 (r, sla, cla, slo, clo, dbr, dbt, dbp)

Arguments

    r
    Geocentric radius in Earth's radii [in]
    sla
    Sine of the geocentric colatitude [in]
    cla
    Cosine of the geocentric colatitude [in]
    slo
    Sine of the longitude [in]
    clo
    Cosine of the longitude [in]
    dbr
    External magnetic field component along "rho" [out]
    dbt
    External magnetic field component along "theta" [out]
    dbp
    External magnetic field component along "phi" [out]

Description

    The subroutine UM532 is an interface between the UNILIB libray and the external magnetic models collected in the package EXT530. The selection of the model and the parameters specific to the model are passed through subroutine UM532 by the argument mext of common block UC140.

Diagnostics

    None

Common Blocks

    UC140, magnetic field description
    UC160, general constants

Dependencies

Called by

    UM530,evaluate the magnetic field vector

Calls

    EXT530, set of external magnetic models:
      MEAD, TSY87S, TSY87L, TSY89, BXYZMU, BDYN, T96_01, BOM97

See also

    None

Reported Bugs

    None

Examples

    None
