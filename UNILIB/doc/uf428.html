UF428

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UF428 - interpolating an altitude

Synopsis 

      REAL*8    falt, bi, ai
      INTEGER*4 ifail 
      
      CALL UF428 (falt, bi, ai, ifail)

Arguments

    falt
    Altitude to be interpolated. [in]
    bi
    Magnetic field intensity at the interpolated location. [out]
    ai
    Altitude at the interpolated location.[out]
    ifail
    Error flag or interpolated location index (see diagnostics). [out]

Description

    The subroutine UF428 interpolates an altitude (falt) along a field line. To be used, the common block UC170 has to include at least two points and the last Runge-Kutta step had to overreach the altitude falt.

    The subroutine UF428 combines linear interpolations and Runge-Kutta steps to determine the correct location of the altitude falt.

    When the subroutine succeeds, the arguments bi and ai are set to the magnetic field intensity and altitude at the interpolated location, respectively. The index of the common block UC170 corresponding to the interpolated location is stored in the argument ifail.

    Note that the subroutine takes into account the step size limit stplst defined in the common block UC190.

Diagnostics

    -42801, altitude is constant
    -42802, too many iterations
    -42803, -42804, problem with the Runge Kutta step
    -42805, boundary condition on the altitude is not between the altitudes of the last points

Common Blocks

    UC160, general constants
    UC170, temporary magnetic field line
    UC190, control parameters, set 1

Dependencies

Called by

    UF422, follow a field line until a boundary condition is reached

Calls

    UF423, Runge Kutta step

See also

    None

Reported Bugs

    None

Examples

    None
