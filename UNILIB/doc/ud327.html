UD327

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UD327 - search an extremum

Synopsis 

      TYPE(zgeo) mgp1, mgp2, mgp3, mextr
      REAL*8        fct1, fct2, fct3, est
      INTEGER*4     ifail

      CALL UD327 (mgp1, mgp2, mgp3, fct1, fct2, fct3, mextr, est, ifail)

Arguments

    mgp1
    First point [in]
    mgp2
    Second point [in]
    mgp3
    Third point [in]
    fct1
    Value at location mgp1 [in]
    fct2
    Value at location mgp2[in]
    fct3
    Value at location mgp3 [in]
    mextr
    Location of the extremum [out]
    est
    Estimation of the function at the extremum [out]
    ifail
    Error flag (see diagnostics) [out]

Description

    The subroutine UD327 makes a parabolic interpolation between three geographic positions and deduces the location of the extremum.

History

    The subroutine UD327 does not exist in version 1.05 and earlier. Errors in the algorithm and its implementation have been fixed in version 2.01. The argument est is not part of the subroutine synopsis in version 2.00 and earlier.

Diagnostics

    -32701, no extremum

Common Blocks

    UC160, general constants

Dependencies

Called by

    UD315, search the mirror point with the lowest altitude

Calls

    None

See also

    None

Reported Bugs

    None

Examples

    None
