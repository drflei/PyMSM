UM512

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UM512 - set the GSFC 12/66 model coefficients

Synopsis 

      REAL*8        tm
      CHARACTER*32  lbl
      
      CALL UM512 (tm, lbl)

Arguments

    tm
    Epoch of the geomagnetic field calculation expressed in year [in]
    lbl
    Label of the geomagnetic field model [out]

Description

    The subroutine UM512 initializes the expansion coefficients of the geomagnetic field inside the common block UC140 with the coefficients of the GSFC 12/66 model. The tm arguments specifies the epoch to which the model has to be extrapolated. The base epoch of the GSFC 12/66 model is 1960. As result, the lbl argument contains a string with the name of the model.

Diagnostics

    None

Common Blocks

    UC140, magnetic field description

Dependencies

Called by

    UM510, select a geomagnetic field model

Calls

    UM515, perform a Schmidt normalisation

See also

    UM511, set the Jensen & Cain 1960 model coefficients
    UM513, set the DGRF/IGRF model coefficients

Reported Bugs

    None

Examples

    None
