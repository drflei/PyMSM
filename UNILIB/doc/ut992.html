UT992

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UT992 - print the magnetic drift shell

Synopsis 

      INTEGER*4     ifail,kunit
      
      CALL UT992 (kunit, ifail)

Arguments

    kunit
    File unit for output (see note of subroutine UT990) [in]
    ifail
    Error flag (see diagnostics) [out]

Description

    The subroutine UT992 prints the set of magnetic field line segments stored in the common blocks UC110, UC120 andUC130.

Diagnostics

    -99201, invalid file unit or no drift shell
    -99202, corrupted field line
    -99203, error on output device

Common Blocks

    UC110, magnetic drift shell description
    UC120, magnetic field line description
    UC130, set of magnetic field line segments
    UC160, general constants

Dependencies

Called by

    None

Calls

    None

See also

    UD310, trace a magnetic drift shell
    UT993, store a magnetic field drift shell

Reported Bugs

    None

Examples

    None
