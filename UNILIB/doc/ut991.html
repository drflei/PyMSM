UT991

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UT991 - print the magnetic field line

Synopsis 

      INTEGER*4 ifail,kunit
      
      CALL UT991 (kunit, ifail)

Arguments

    kunit
    File unit for output (see note of subroutine UT990) [in]
    ifail
    Error flag (see diagnostics) [out]

Description

    The subroutine UT991 prints the magnetic field line segment stored in the common block UC170.

Diagnostics

    -99101, no file unit specified or no current field line
    -99102, corrupted field line
    -99103, error on output device

Common Blocks

    UC160, general constants
    UC170, temporary magnetic field line

Dependencies

Called by

    None

Calls

    None

See also

    UF420, trace a magnetic field line segment passing through a given position

Reported Bugs

    None

Examples

    G.06 How to start using the UNILIB library ?
