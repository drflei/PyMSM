UD331

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UD331 - evaluate the magnetic flux through a spherical cap

Synopsis 

      REAL*8    colat,phi
      INTEGER*4 ifail
      
      UD331 (colat, phi, ifail)

Arguments

    colat
    Pole selection [in]
    phi
    Third invariant [out]
    ifail
    Error flag (see diagnostics) [out]

Description

    The subroutine UD331 computes the magnetic flux passing through a spherical cap. The spherical cap is defined by the set of northern or southern foot points of the magnetic drift shell stored in the common blocks UC110, UC120 and UC130.

    When the argument colat is set to 0, the subroutine makes the integration over the north cap and uses the arguments mfl().footpn of common block UC120. When the argument colat is set to 180, the subroutine makes the integration over the south cap and uses the arguments mfl().footps of common block UC120.

Diagnostics

    -33101, invalid drift shell description
    -33102, illegal value of argument colat

Common Blocks

    UC110, magnetic drift shell description
    UC120, magnetic field line description

Dependencies

Called by

    UD330, evaluate the third invariant

Calls

    UD332, evaluate the magnetic flux through a spherical pie

See also

    None

Reported Bugs

Examples
