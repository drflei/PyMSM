UA612

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UA612 - provide a list of atmospheric constituents

Synopsis 

      INTEGER*4     ksp(30),n,ifail
      CHARACTER*(*) lbl
      
      CALL UA612 (ksp, n, lbl, ifail)

Arguments

    ksp
    List of species number [in]
    n
    Number of species [in]
    lbl
    Output string [out]
    ifail
    Error flag (see diagnostics) [out]

Description

    The subroutine UA612 returns a string in argument lbl containing the list of the species set in the argument ksp. The valid values for argument ksp are listed in the table below.

    1 = H 11 = CO2 21 = He+
    2 = He 12 = NH3 22 = C+
    3 = Ne 13 = CH4 23 = N+
    4 = Ar 14 not used 24 = O+
    5 = Kr 15 not used 25 = Ne+
    6 = Xe 16 = O. 26 = N2+
    7 = H217 = N. 27 = NO+
    8 = N218 not used 28 = O2+
    9 = O219 = e- 29 not used
    10 = CO 20 = p+ 30 not used

Diagnostics

    -61201, invalid species number

Common Blocks

    None

Dependencies

Called by

    UA610, select an atmospheric, ionospheric and/or plasmaspheric model

Calls

    None

See also

    UC150, atmospheric model description

Reported Bugs

    None

Examples

    None
