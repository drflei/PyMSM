UM524

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UM524 - GEO to SM transformation

Synopsis 

      INTEGER*4 kunit
       
      CALL UM524 (kunit)

Arguments

    kunit
    File unit for the summary table (see note of subroutine UT990) [in]

Description

    The subroutine UM524 computes the transformation array from GEO coordinate to SM coordinate as well as the geomagnetic dipole tilt angle. The results are stored in the arguments mext.trans and mext.tilt of the common block UC140. As input, the subroutine uses the position of the Sun and the orientation of the magnetic dipole axis stored in the arguments msun and mint of the common block UC140, respectively. Note that the Sun position can be easily modified by the use of subroutine UM522.

    When kunit is greater than zero, a summary table is written by the subroutine.

Diagnostics

    * The program is aborted when an error on the output device

Common Blocks

    UC140, magnetic field description
    UC160, general constants

Dependencies

Called by

    UM520, select an external magnetic field model
    UM539, evaluate the magnetic field

Calls

    None

See also

    UM523, GEO to GSM transformation

Reported Bugs

    None

Examples

    #2, evaluation of the (B, L) coordinates
