Question

    G.07 - Does a similar software or library exist ?

Answer

    There exist softwares, source codes and services that provide the same kind of features as the UNILIB libray. An non-exhausive list is given below. The list has been divided in two parts:
    1. the interactive services which allow to run models through the World Wide Web (WWW) with the help of FORM pages;
    2. the software and source codes which need to be downloaded and execute on the user's computer.
    Note that the library is part of the second category.

Interactive services

    Space ENVironment Information System (SPENVIS)
    SPENVIS is the ESA system of standardized access to models of the space environment. It includes particle radiation models, radiation effects models, spacecraft charging, atmosphere models, debris and meteroid models. The model results are returned in the form of tabular and graphical outputs. The magnetic field package of SPENVIS includes magnetic field lines and drift shell tracing and visualisation.
    Community Coordinated Modelling Center (CCMC)
    The CCMC is a multi-agency partnership to enable, support and perform the research and development for next-generation space science and space weather models and is hosted at NASA's Goddard Space Flight Center. It provides a simple access to the MSIS, IRI, IGRF and Tsyganenko T96 and other models. The site includes also useful information about the different models.

Softwares and source codes

    IRBEM
    The International Radiation Belt Environment Modeling (IRBEM) library is freely distributed under the umbrella of the PRBEM COSPAR panel under a LGPL license. The library allows to compute magnetic coordinates and fields for any location in the Earth's environment for various magnetic field models, coordinate transformation and evaluate particle fluxes from various specification models.
    UNIRAD
    UNIRAD is a software package developed by BISA for ESA to evaluate the radiation fluences and doses expected in a spacecraft for a mission defined by given orbital characteristics. Its BLXTRA package includes an access to magnetic field models and geomagnetic coordinates. The software is distributed in the form of executables for different platforms and operating systems. The distribution includes a graphics package based on IDL routines.
    Community Coordinated Modelling Center (CCMC)
    The CCMC provides a rich catalog of models.
