UT980

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UT980 - print the error messages

Synopsis 

      INTEGER*4     ifail, kunit, kfail
      CHARACTER*(*) lcall
      
      CALL UT980 (lcall, kfail, kunit, ifail)

Arguments

    lcall
    Subroutine name [in]
    kfail
    Error code [in]
    kunit
    File unit for output message (see note of subroutine UT990) [in]
    ifail
    Error flag (see diagnostics) [out]

Description

    The subroutine UT980 provides basic information on error flags returned by a subroutine of the UNILIB libray. The subroutine takes as input the name of a subroutine (e.g. UM530) and an error code (e.g. -53101).

History

Since the diagnostic data base was out of date, the subroutine UT980 has been reviewed in version 2.01, and, the error messages have been simplified.

Diagnostics

    -98001, invalid value of the argument lcall
    -98002, invalid error code
    -98004, error with output device
    -98005, invalid value of argument lcall
    -98006, incorrect name of subroutine
    -98007, error on output device
