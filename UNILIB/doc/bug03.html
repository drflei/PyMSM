Bug Report

[ Signature | Diagnostic | Solution | History ]

Signature

    R.03 - Malfunction of subroutine UF426

Diagnostic

    A small bug was present in the algorithm of subroutines UF426, UF427 and UF428 and was highlighted when the global precision of the drift shell tracing has been increased. Due to this bug, the error code -41110 was anomalously returned by the subroutine UD310. The error code was consecutive to an error code return by the subroutine UF423 when it is called by the subroutine UF426 with an abnormally small value of the argument scale.

Solution

    The algorithms of the three subroutines have been slighty modified. To illustrate the effect of the correction, the result obtained for the evaluation of the mirror-point lowest altitude of different drift shells are compared in the table below for the versions 1.11 and 1.12, respectively. See FAQ T.06 for detailled description of these evaluations.
