UM515

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UM515 - perform a Schmidt normalisation

Synopsis 

      REAL*8    dt, g0(nx140,nx140), g1(nx140,nx140), 
                    g2(nx140,nx140), g(nx140,nx140)
      INTEGER*4 nmax      
      CALL UM515 (dt, nmax, g0, g1, g2, g)

Arguments

    dt
    Time in year for the extrapolation [in]
    nmax
    Effective size of array g0, g1, g2 and g [in]
    g0
    Expansion coefficients at the base epoch [in]
    g1
    First derivative of the expansion coefficients [in]
    g2
    Second derivative of the expansion coefficients [in]
    g
    Extrapolated and normalized expansion coefficients [out]

Description

    The subroutine UM515 extrapolates the expansion coefficients of a geomagnetic field model and applies the Schmidt normalisation. The dt argument specifies the time delay between the base epoch of the model and the epoch for which the model has to be extrapolated. As result, the g argument contains a array with the new expansion coefficients.

Diagnostics

    None

Common Blocks

    None

Dependencies

Called by

    UM512, set the GSFC 12/66 model coefficients
    UM513, set the DGRF/IGRF model coefficients
