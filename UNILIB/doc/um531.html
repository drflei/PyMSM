UM531

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UM531 - geomagnetic field

Synopsis 

      REAL*8    rkm, st, ct, sph, cph, br, bt, bp
      INTEGER*4 ifail
      
      CALL UM531 (rkm, st, ct, sph, cph, br, bt, bp, ifail)

Arguments

    rkm
    Geodetic radius in Earth's radii [in]
    st
    Sine of the geodetic colatitude [in]
    ct
    Cosine of the geodetic colatitude [in]
    sph
    Sine of the longitude [in]
    cph
    Cosine of the longitude [in]
    br
    Geomagnetic field component along "rho" [out]
    bt
    Geomagnetic field component along "theta" [out]
    bp
    Geomagnetic field component along "phi" [out]
    ifail
    Error flag (see diagnostics)[out]

Description

    The subroutine UM531 evaluates the geomagnetic field vector from the Schmidt-normalized coefficients stored in argument mint of common block UC140.

History

    In version 2.03 and earlier, the error diagnostic -53101 is generated for both error diagnostics -53101 and -53102.

Diagnostics

    -53101, order of the expansion exceeds 14
    -53102, too small value of the sine of the geodetic colatitude

Common Blocks

    UC140, magnetic field description

Dependencies

Called by

    UM530, evaluate the magnetic field vector

Calls

    None

See also

    UM515, perform a Schmidt normalisation
    UM537, Kluge evaluation of the geomagnetic field

Reported Bugs

    None

Examples

    None
