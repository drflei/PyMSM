UL230

[ Name | Synopsis | Arguments | Description | Diagnostics | Common Blocks | Dependencies | See Also | Reported Bugs | Examples ]

Name

    Subroutine UL230 - evaluate the integral invariant coordinate I

Synopsis 

      REAL*8    finv
      INTEGER*4 ifail
      
      CALL UL230 (finv, ifail)

Arguments

    finv
    Integral invariant coordinate [out]
    ifail
    Error flag (see diagnostics) [out]

Description

    The subroutine UL230 evaluates the integral invariant coordinate I for the current magnetic field line segment stored in the common block UC170. The integration algorithm takes advantage from the fact that the magnetic field line segment has been traced by the fourth-order Runge-Kutta method. The integral invariant coordinate I is mainly used to determine McIlwain's shell parameter L.

References

  • Heynderickx, D., and Lemaire, J., Improvements to trapped radiation software, TREND, Technical Note 1 (1993)

Diagnostics

    -23001, field line segment corrupted
    -23002, different B at the field line segment extremities
    -23003, invalid B along the field line segment

Common Blocks

    UC170, temporary magnetic field line
    UC190, control parameters, set 1

Dependencies

Called by

    UL220, get information on a magnetic field line segment
    UF411, search a local magnetic equator
    UF415, rebuild the labels of the field line

Calls

    None

See also

    UL240, evaluate the Hilton's function
    UF420, trace a magnetic field line segment passing through a given position

Reported Bugs

    None

Examples

    None
